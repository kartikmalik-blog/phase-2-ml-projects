# 🚀 Machine Learning Projects — Phase 2

This repository contains my **advanced machine learning projects**, where I moved from basic concepts to more **real-world problem solving and model optimization**.

In this phase, I focused on:
- Handling real datasets
- Model evaluation using multiple metrics
- Dealing with class imbalance
- Cross-validation & model tuning
- Avoiding overfitting

---

## 📂 Projects

### 💳 Fraud Detection Engine
- **Goal:** Detect fraudulent credit card transactions
- **Challenge:** Highly imbalanced dataset
- **Focus Metric:** Recall (minimizing false negatives)
- **Techniques:**
  - Logistic Regression
  - Random Forest
  - XGBoost
- **Outcome:** Optimized for detecting fraud cases reliably

---

### 🚀 Rocket Launch Predictor
- **Goal:** Predict success or failure of rocket launches
- **Techniques:**
  - XGBoost (Best model)
  - Cross-validation
- **Key Insight:** Initial perfect accuracy indicated overfitting → improved with proper validation
- **Final Focus:** Realistic model evaluation (AUC, Recall)

---

### 🌟 Star Classification Engine
- **Goal:** Classify celestial objects (Stars, Galaxies, Quasars)
- **Dataset:** SDSS astronomical dataset
- **Techniques:** Multiple classification models
- **Outcome:** High accuracy classification with feature importance analysis
